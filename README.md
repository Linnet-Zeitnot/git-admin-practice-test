# Практика Git

## Что нужно сделать

1. В истории должны быть коммиты с сообщениями:
   - `initial commit (windows)`
   - `feature change (windows)`
   - `feature change (ubuntu)`
   - `merge ubuntu into main`

2. Должны существовать ветки:
   - `feature-windows`
   - `feature-ubuntu`

3. Обе feature-ветки должны быть слиты в основную ветку (`main` или `master`).

4. Заполнить файл `proof/windows.txt` выводом команд из Windows.

5. Заполнить файл `proof/ubuntu.txt` выводом команд из Ubuntu.

6. Добавить в этот README строку:

```

STUDENT_TOKEN: TOKEN-<ваш_github_username>

```

Пример:

```

STUDENT_TOKEN: TOKEN-ivanov-io

```

## Проверка

1. Откройте вкладку **Actions**.
2. Последний запуск должен быть зелёным.
3. В разделе **Summary** появится код вида:

```

TB-XXXXXXXXXXXX

```
